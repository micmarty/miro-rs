# miro-rs [work-in-progress]
Based on Miro REST API reference: https://developers.miro.com/reference


# How to use
Add this to your `Cargo.toml` file
```toml
[dependencies]
miro-rs = { git = "https://github.com/micmarty-rs/regex", branch = "master" }
```
